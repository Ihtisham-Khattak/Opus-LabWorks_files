Getting Started

The following section explains how to set up the Flutter template.

1. Download and setup Flutter

  • Get the Flutter SDK from the following link: https://docs.flutter.dev/get-started/install

  • Once downloaded and install, verify the installation of Flutter SDK & Android SDK, by runnning the following command in terminal: 

    flutter doctor

2. Setup IDE

  • Download and install Android Studio from the following link: https://developer.android.com/studio

  • Start Android Studio. Open plugin preferences (Preferences > Plugins as of v3.6.3.0 or later). Select the Flutter plugin and click Install. Click Yes when prompted to install the Dart plugin. Click Restart when prompted. For more info, visit https://docs.flutter.dev/get-started/editor?tab=androidstudio

  • Or install Visual Studio Code from the following link: https://code.visualstudio.com/

  • Start VS Code. Invoke View > Command Palette. Type “install”, and select Extensions: Install Extensions. Type “flutter” in the extensions search field, select Flutter in the list and click Install. This also installs the required Dart plugin. For more info, visit: https://docs.flutter.dev/get-started/editor?tab=vscode

3. Run the app

  • Open Android Studio, Go to the File menu, click on Open

  • Then select the flutter project and click on open.

  • Go to the lib folder in the project

  • Locate the main Android Studio toolbar

  • In the target selector,, select an Android device for running the app. If none are listed as available, select Tools> Android > AVD Manager and create one there. For details, see Managing AVDs

  • Click the run icon in the toolbar, or invoke the menu item Run > Run

Customizations

The following section explains how to do the modifications in the template/app.

1. Change app icon of template/app

  • To change the app icon, change the app_icon.png by your image. You need to resize your image (1024 x 1024)

  • Now run the following commands to apply your app icon: 

    flutter pub get
    flutter packages pub run flutter_launcher_icons:main

  • For more info, visit the official page of this package https://pub.dev/packages/flutter_launcher_icons

2. Change app splash of template/app

  • To change the splash, change the app_splash.png by your image. You need to resize your image (2732 x 2732).

  • Now run the following commands to apply your app icon: 

    flutter pub get
    flutter packages pub run flutter_launcher_splash:create

  • For more info, visit the official page of this package: https://pub.dev/packages/flutter_native_splash

Platform Builds

The following section explains how to make a build for Android and iOS.

1. Android setup and generate build (.apk)

  • Ensure you have set up Android SDK properly.

  • To build a debuggable .apk file, run the following command: 

    flutter build apk –debug

2. iOS setup and generate build

  • Ensure you have Xcode set up properly: https://apps.apple.com/us/app/xcode/id497799835

  • Run the following command to build an Xcode project: 

    flutter build ios

  • Open project in Xcode by opening .xcworkspace file in ./ios/Runner.xcworkspace

  • While launching the app from mac, if you see
    ERROR: Could not connect to lockdownd, error code -17,
    make sure that you have trusted your computer.
